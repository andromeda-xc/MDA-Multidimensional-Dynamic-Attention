# MDA-Multidimensional-Dynamic-Attention
This repository contains the implementation of Multidimensional Dynamic Attention (MDA) — a novel deep learning framework designed to enhance forecasting performance on multivariate time series data by dynamically capturing both temporal and inter-dimensional dependencies.
